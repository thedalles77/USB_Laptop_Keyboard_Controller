This remix is based on the fantastic design from the GreenMamba and uses my 
Teensy USB Controller board documented in step 17 of my Instructable found here:
https://www.instructables.com/How-to-Make-a-USB-Laptop-Keyboard-Controller/
The files noted as Original have simply been cut in left and right halves to 
fit my printer. These are the files I used to print the pictured keyboard 
enclosure. I had to drill new mounting holes for the Teensy 4.0 connector board 
which has a slightly different hole spacing than the obsolete Teensy 3.2 board 
that was used by the GreenMamba. When I glued the left and right bases together, 
I added an extra support piece for strength. I didn't want the USB hub so there 
is a patch that plugs the holes in the back. My USB cable was skinny so I added 
an extra piece to the Kabel_Holder to pinch the cable.

For anyone starting from scratch, use the files with T4.0 in the name. These 
will have the correct mounting hole spacing for the Teensy 4.0 connector board(no drilling needed). 
Choose whether you want the USB hub or not for the base right half. I made no changes 
to the bezel although I recommend installing knurled threaded brass inserts using a 
heat set insert. I just filled the bezel  mounting holes with epoxy and then drilled 
small holes that a screw could cut threads into (not pretty).
