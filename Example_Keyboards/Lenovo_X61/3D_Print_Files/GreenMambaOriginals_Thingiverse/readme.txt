Summary
ThinkBoard
This is a keyboard made from the great article by Frank Adams on instructables.

The PCB, the code and great documentation can be found in the article above.
I used a X61 keyboard (FRU: 42T3037) and the connector and software is compatible with the T61 mentioned in the article. 
For the German layout, please see my notes in the "I made this" section of the article.
I ordered the connector mention in the article
The PCB was produced by a regular PCB manufacturer (JLCPCB, PCBWAY,...) without modification to the original files from 
the article and worked perfectly.
The USB-hub is a "AUKEY USB 3.0 Hub 4 Port Ultra Slim" which I bought on amazon, but I'm pretty sure, you can find the 
same one under another brand, or modify the attached files.
The screws are M3 inbus for the case and M2 for the keyboard itself. Be careful when tightening! They only have a few 
if not less) millimeter to grab.
STEP, IGES and f3z (fusion360) files are included.
The weights are for balancing tires on cars. You can find them for dirt cheap on the web.
